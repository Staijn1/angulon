# angulon
Control your leds over serial using electron &amp; angular

# Getting started with source code
Run ```npm install```  
After running, run ```npm start```


#Features
- Audio Visualizer
- Wide variety in modes
- Saving settings and current colors to file
- Modern UI
- Easy to use
